# Two Sum

---

## Discription

You are given two integers **A** and **B**.  
Your task is to compute and output their sum.

---

## Input Format

The input consists of two integers separated by space:

```
A B
```

---

## Output Format

Output a single integer representing:

```
A + B
```

---

## Constraints

```
-1000 ≤ A, B ≤ 1000
```

---

## Example

### Input
```text
2 3
```

### Output
```text
5
```

---

## Explanation

Given the input:

```
A = 2
B = 3
```

The sum is:

```
A + B = 5
```

So the output is:

```
5
```
