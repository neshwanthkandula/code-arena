# Two Sum

---

## Discription

find the element in the given two numbers

---

## Input Format

The input consists of two integers separated by space:

```
A B
```

---

## Output Format

Output a single integer representing:

```
maximum(A, B)
```

---

## Constraints

```
-1000 ≤ A, B ≤ 1000
```

---

## Example

### Input
```text
2 3
```

### Output
```text
3
```

---

## Explanation

Given the input:

```
A = 2
B = 3
```

The maximum is:

```
maximum(2, 3) = 3
```

So the output is:

```
3
```
